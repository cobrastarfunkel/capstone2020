NetCat Backdoor

C:\ > The following Scenario is going to walk you through the process of establishing a persistent connection to a remote server using a netcat backdoor.

C:\ > The scenario is:

  • You have received an email from the "IT department" instructing you to download an update.
  • This update will send a reverse shell back to the attackers computer.
  • The reverse shell will be caught by Metasploit and used to install netcat.
  • The shell will also allow the attacker to modify registry keys to ensure a connection in the future.

C:\ > In the next screen you will see a simulation of an email service. Click the IT departments email and run the attachment to get started.

This Scenario is interactive, but no modifications will be made to your computer.
